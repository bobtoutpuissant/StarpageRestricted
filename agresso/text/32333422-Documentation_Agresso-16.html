AGRESSO
      2. Achats
          2.4. Demande d'achat ou demande de réapprovisionnement
 2.4.1. Création de la DA par le demandeur 

Pour créer une DA (appelée Demande de réapprovisionnement dans Agresso), cliquer sur "Achats" puis "Demande simple"

L'écran est divisé en 3 parties :

  1. les informations générales
  2. les articles que l'on va commander
  3. les informations complémentaires de chaque ligne demandée

A noter que vosu pouvez sauvegarder votre DA quand vous voulez en cliquant sur "Sauve"

1 les informations générales

  • le nom du demandeur apparait (c'est le votre !)
  • le fournisseur
    • pour rechercher un fournisseur, saisissez son nom ou une partie de son nom
      • exemple DELL me donne 2 réponses qui contiennent DELL dans leur nom
    • une fois le fournisseur trouvé, validez (tabulez)
    • on peut choisir quelle adresse du fournisseur apparaitra sur la commande achat
      • si on ne sait pas, le service achat complétera
  • le lieu de livraison
    • choisissez le lieu dans la fenêtre déroulante
    • si le lieu n'existe pas, contacter le support AGRESSO de Telespazio (L. BABILLIOT)

2 les articles

  • le code article
    • saisissez le code si vous le connaissez
    • sinon recherchez le code en saisissant une partie de sa description
      • exemple : saisissez "computer" et vous verrez tous les articles comportant ce nom
    • une fois le nom trouvé, tabulez pour valider
  • la description
    • la description par défaut du code article apparait
    • vous pouvez la modifier
    • sachez aussi qu'une description "longue" de votre produit est possible (voir ci-dessous)
  • la quantité
    • si le champ est grisé, la notion de quantité n'est pas importante
    • sinon saisissez la quantité (par défaut, la valeur est à 1)
  • le montant
    • il s'agit toujours d'un montant HORS TAXES
    • si vous le connaissez saisissez le sinon demandez au service achat un prix
  • le délai
    • très important pour les contrôleurs de BU pour suivre les "engagements achats"
    • si vous avez 3 délais différents vous aurez donc 3 lignes d'article

3 les informations complémentaires

  • vous pouvez saisir des informations complémentaires pour chaque article
    • pour cela cliquer sur la double flêche qui va vers le bas
    • a noter que le copier coller d'un texte de votre PC fonctionne
  • vous pouvez aussi saisir des pièces jointes
    • pour les joindre cliquer sur le trombone en haut à droite de l'écran
    • il existe 2 types de pièces jointes
      • celles que vous voulez joindre à la commande
      • celles que vous voulez joindre à la DA mais qui ne seront pas transmises dans la commande
    • vous pouvez noter quelque chose pour chaque pièce jointe

Voilà, ah oui, au fait, n'oubliez pas de sauvegarder de temps en temps, on ne sait jamais
