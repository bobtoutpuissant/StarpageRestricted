AGRESSO
      2. Achats
 2.3. Catégories d'articles 

La base des achats est constitué :

  • de demandes d'achat
  • de commandes

Il existe 3 catégories de produits

  • les composants sur étagères ou COTS (ce sont des composants dits "standard", sans spécification)
  • les achats de type "intellectuels" ou complexes (il va s'agir de commandes de prestations intellectuelles, d'intérim, ou de  composants complexes ou entièrement sous-traités)
  • les achats de produits avec abonnement, achetés à l'année, type loyer, téléphonie, électricité, etc.

Pour ces 3 types de produits, nous aurons 3 numérotations de commandes :

  • pour les COTS, les commandes seront numérotées 115xxxxxxx
  • pour les achats "intellectuels" ou complexes, les commandes seront numérotées 515xxxxxx
  • pour les achats avec abonnement, les commandes seront numérotées 715xxxxxx

Aussi, la responsabilité du délai d'achat de ces types d'achat sera différente

  • pour les COTS, les acheteurs peuvent maitriser le délai
  • pour les achats "intellectuels" ou complexes, les services techniques maitrisent le délai
  • pour les achats avec abonnement, la notion de délai n'est pas importante
